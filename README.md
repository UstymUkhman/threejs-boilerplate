## Download ##

`git clone https://github.com/UstymUkhman/threejs-boilerplate.git`

`cd threejs-boilerplate`

## Develop ##

`pnpm i`

`pnpm start`

## Lint ##

`pnpm lint:js`

`pnpm lint:css`

## Test ##

`pnpm test`

`pnpm test:watch`

`pnpm test:cover`

## Build ##

`pnpm build`

`pnpm serve`
